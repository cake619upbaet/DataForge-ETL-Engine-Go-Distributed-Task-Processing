## 📋 What is DataForge-ETL-Engine-Go-Distributed-Task-Processing?

DataForge-ETL-Engine-Go-Distributed-Task-Processing is a tool that helps you move, clean, and organize large amounts of data. It works by breaking down data tasks into smaller jobs and handles them in parallel across multiple computers or cores. This lets you process data quickly and with less effort.

The engine is built to work with data pipelines. It can import data from different sources, normalize formats, remove duplicate entries, and clean the data. It is designed for users who need to handle heavy data loads but do not want to spend hours on manual processing.

## ⚙️ How to Use DataForge

DataForge focuses on simplicity to help you process data without programming.

### Creating a New ETL Task

1. Click the **New Task** button.
2. Select the data source: choose between file types (CSV, JSON) or database connections.
3. Configure the import settings, such as file location or database address.
4. Select cleaning options: remove nulls, fix formats, trim spaces.
5. Choose normalization rules to unify all data fields.
6. Enable deduplication to remove repeated entries.
7. Set the destination where cleaned data will save.
8. Click **Run** to start processing.

You can watch task progress live, including number of records processed and any errors.

### Managing Distributed Processing

DataForge can split tasks across multiple machines or cores:

- In the **Settings** menu, add IP addresses of other computers on your network.
- Assign each a percentage share of the total task.
- The program balances workload automatically and monitors each node.

This helps scale up processing when working with very large datasets.

## 🛠️ Common Troubleshooting

- **Installer Won't Run:** Make sure you have administrator rights and the downloaded file is not blocked by Windows Defender.
- **Slow Processing:** Check your CPU and RAM usage. Close unnecessary programs to free resources.
- **Connection Errors:** Verify your network settings if using distributed task processing.
- **Data Import Fails:** Confirm the file paths and database access credentials.
